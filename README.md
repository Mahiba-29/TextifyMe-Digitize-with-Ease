# TextifyMe-Digitize-with-Ease
TextifyMe – Digitize with Ease (BE Final Year Project, MSIC Winner | 2023–2024) An intelligent text digitization platform that converts handwritten content into digital text, provides summarization and multilingual translation using transformer-based ML models, with a simple, accessible web interface built using HTML, CSS, and Bootstrap.
